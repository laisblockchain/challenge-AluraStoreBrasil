# AluraStoreBrasil - Análise de Dados

## Propósito da Análise
Este projeto foi desenvolvido como parte de um challenge de Data Science da Oracle + Alura. O objetivo é analisar dados de vendas de quatro lojas da Alura Store Brasil para identificar qual delas apresenta o pior desempenho e recomendar qual loja o Senhor João deveria vender. A análise considera métricas como faturamento total, avaliações dos clientes, categorias de produtos mais vendidas, produtos mais populares e frete médio.

## Estrutura do Projeto
O projeto está organizado da seguinte forma: `AluraStoreBrasil.ipynb` contém todo o processo de análise de dados, incluindo carregamento dos dados, exploração, criação de gráficos e conclusão; a pasta `dados/` contém os arquivos de dados das quatro lojas (`loja_1.csv`, `loja_2.csv`, `loja_3.csv`, `loja_4.csv`); e o arquivo `README.md` apresenta a documentação do projeto.

## Exemplos de Gráficos e Insights
Foram criadas visualizações para facilitar a análise dos dados, como gráficos de barras para comparar o faturamento entre as lojas, gráficos para visualizar o frete médio e gráficos para analisar as categorias de produtos mais vendidas. Os principais insights foram: a Loja 1 apresentou o maior faturamento, a Loja 3 teve a melhor avaliação média dos clientes, a Loja 4 possui o menor frete médio, porém também apresentou o menor faturamento geral, indicando desempenho inferior em comparação com as demais lojas.

## Instruções para Executar o Notebook
1. Clone o repositório: `git clone https://github.com/seu-usuario/AluraStoreBrasil.git`  
2. Acesse a pasta do projeto: `cd AluraStoreBrasil`  
3. Instale as bibliotecas necessárias: `pip install pandas matplotlib`  
4. Abra o Jupyter Notebook: `jupyter notebook`  
5. Execute o arquivo `AluraStoreBrasil.ipynb` para visualizar toda a análise.
